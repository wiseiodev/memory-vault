This is a Vercel-first personal memory app monorepo.

Use `pnpm` workspaces.

Common repo commands:
- `pnpm check`
- `pnpm check:ci`
- `pnpm qc`

Work is tracked in Linear; prefer the `linear` / `li` CLI for Linear operations.
